📌 How to Use

🚀 QUICK START:

  1. Select your Destination Airport above.
  2. Place your phone near the window.
  3. Keep the screen ON (Do not lock the phone).

✅ OFFLINE SETUP (100% Reliable):

Open this app via a web link (HTTPS). Tap the browser's Share/Menu button and select "Add to Home Screen". Do not run as a local HTML file on Android, as GPS will be blocked.

✈️ Status Info:

  • Parked: Distance locked to stop GPS jumping.
  • Weak Signal: No GPS signal = No updates.
  • Blinking Numbers: Estimated mode (hardware lost speed data).
